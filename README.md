# Fetch Holiday

GraphQL interface in Express (NodeJS) for fetching holidays information from [Calendarific](https://calendarific.com/)

## Run server

1. Clone repo

  ```
  git clone https://github.com/tracy2811/fetch-holiday.git
  cd fetch-holiday
  ```

2. Run server

  * On Local Machine

    ```
    node dist/server.js
    ```

  * Using Docker

    ```
    docker-compose up
    ```
